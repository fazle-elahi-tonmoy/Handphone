# HandPhone – Arduino Based Emergency Calling Device

<img src="images/Handphone_User_Demo.jpg" width="700"> 

A wearable Arduino-based emergency communication device designed to trigger quick distress calls and alerts through a simple hand gesture interface.

The project demonstrates a low-cost wearable communication concept for:

* 🚨 Emergency assistance
* 🧓 Elderly support systems
* ♿ Accessibility devices
* 🆘 Quick distress signaling
* 📡 Wearable embedded systems

The device is lightweight, portable, and designed to be worn directly on the hand for rapid activation.

---

# 📞 Project Overview

**HandPhone** is a wearable emergency communication prototype that allows users to trigger alerts or calls using hand-mounted controls.

The system is designed to provide:

* Fast emergency activation
* Simple wearable interaction
* Portable operation
* Easy accessibility

The project demonstrates how embedded systems can be integrated into wearable safety technology.
